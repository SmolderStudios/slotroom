# SLOTROOM

A PS1/PSX-style horror slot-machine game (walk a grimy room, gamble to survive,
dig your way out). Playable in the browser via GitHub Pages — no install needed.

**Play:** https://smolderstudios.github.io/slotroom/

Single self-contained `index.html` (the built game) + `models/` (Tripo .glb assets).
Source lives locally in `C:\code\slotroom` and is rebuilt with `node build.js`.
